# Mi Proyecto de JavaScript

Una breve descripción de lo que hace este script (por ejemplo: "Ejemplos prácticos sobre operadores de comparación y conversión de tipos en JavaScript").

## 🚀 Características
* Demostración de desigualdad suelta (`!=`).
* Demostración de desigualdad estricta (`!==`).
* Explicación visual de la coerción de tipos.

## 🛠️ Requisitos
* [Node.js](https://nodejs.org) instalado (versión 16 o superior recomendada).

## 💻 Instalación y Uso
1. Clona este repositorio o descarga los archivos.
2. Abre tu terminal en la carpeta del proyecto.
3. Ejecuta el código con el siguiente comando:
   ```bash
   node index.js
   ```
